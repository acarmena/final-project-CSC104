The Emerald Grove Website
🌿 Overview
The Emerald Grove is a custom-designed café website built using HTML, CSS, and static assets.
This project recreates a modern, elegant landing page featuring:

A deep-green botanical theme

Minimalist navigation

A centered, serif-style brand title

A decorative branch illustration

Contact information

Social media icons

Separate pages for Menu, Inside the Café, About Us, and Contact
-- Menu:
--- shows what items are available and their prices.

-- Inside the Cafe:
--- shows and describes the interior and ambiance the cfe wishes to exert

-- About and Contact:
--- Shows the sense of community and coziness the cafe wishes to make the customers feel.

--- The contact tells of the cafe's fake number, email, and address.

The design aims to capture the calming, tropical ambiance represented in the café brand.

Technologies Used
HTML5 for content structure

CSS3 for styling, layout, and typography

Google Fonts (Cormorant) for an elegant serif aesthetic

ChatGPT/ VS Chat for stylization help and readme file information

VS Code + Live Server for development and live preview

🎨 Design Notes
The overall color scheme is a deep green (#2E4635) with cream text for contrast.

The typography uses Cormorant from Google Fonts for a classic, botanical feel.

The right-aligned branch illustration is placed with soft opacity to match the aesthetic.

Navigation uses centered, spaced text for an upscale visual balance.

Social icons sit in the footer and fade slightly when hovered over.

✏️ Customization
You can easily update:

Colors — change in style.css under body, nav, and .hero.

Logo — replace images/logo.png.

Illustration — replace images/Website_Outline.png.

Fonts — modify the Google Fonts import inside the <head> section.
